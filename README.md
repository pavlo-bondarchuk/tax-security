# Tax Security 2026

Responsive landing page prototype for an online tax law intensive by GK & Partners.

## Live preview

https://pavlo-bondarchuk.github.io/tax-security/

## Repository

https://github.com/pavlo-bondarchuk/tax-security

## Features

- Responsive one-page layout
- English interface and accessibility attributes
- Speaker and audience photography
- YouTube Shorts testimonials
- Pricing section and lead form prototype
- GitHub Pages deployment

## Technology

- HTML5
- CSS3
- Vanilla JavaScript

## Deployment

The site is published from the `main` branch through GitHub Pages. Any commit to `main` triggers the standard Pages build and deployment process.

## Project status

Prototype. The form currently demonstrates client-side validation and a success modal but is not connected to a backend or CRM.
